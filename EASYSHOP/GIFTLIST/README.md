Objetivo do módulo desenvolver uma lista de casamento

A pessoa que compra o produto ele não compra efetivamente o produto mas sim o crédito para o presenteado, para que o mesmo possa trocar esses presente por outros produto, quando resgatado é gerado um pedido com status lista de casamento dado como pago. Importante ressaltar que todo o valor do pedido, valor do carrinho e frete é debitado do saldo do cliente.
