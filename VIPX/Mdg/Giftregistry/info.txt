Cliente possui um e-commerce de marmitas, e solicitou que fosse desenvolvido algo para possibilitar o cliente de montar o seu próprio prato, mas os igredientes mudam de acordo com o prato selecionado. Para resolver esse desafio, utilizei um produto simples, com opções personalizadas, mas os campos não são ligados com isso criar um campo extra para ligar os inputs e assim poder trabalhar com jquery. 

Objetivo do módulo 
 Criar um input para ligar os campos personalizados para ser possível manipulalos via jquery
 Criar dinamização dos inputs de acordo com um input select selecionado

Obs. Utilizei como esqueleto a estrutura do módulo MDG. Por isso não tem minha namespaces
